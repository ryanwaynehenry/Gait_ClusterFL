# ClusterFL
This is the repo for MobiSys 2021 paper: "ClusterFL: A Similarity-Aware Federated Learning System for Human Activity Recognition".

<br>

# Requirements
The program has been tested in the following environment: 
* Ubuntu 18.04
* Python 3.6.8
* Tensorflow 2.4.0
* sklearn 0.23.1
* opencv-python 4.2.0
* Keras-python 2.3.1
* numpy 1.19.5
